# Hemanth Kumar Konda Portfolio

A modern, Apple-inspired static portfolio website for Hemanth Kumar Konda.

## How to use

1. Replace `assets/profile-placeholder.svg` with Hemanth's real cutout profile image.
   - Recommended filename: `profile-placeholder.svg` if you want no code changes.
   - Better production format: WebP or AVIF.
2. Deploy the full folder to GitHub Pages, Netlify, Vercel, or any static host.

## GitHub Pages deployment

- Create a repository.
- Upload all files from this folder.
- Go to Settings → Pages.
- Set source to the main branch and root folder.
- Save and open the published site URL.

## Files

- `index.html` — semantic structure, SEO, schema.org Person markup, page content.
- `styles.css` — Apple-inspired visual system, spacing, cards, timeline, responsive design.
- `script.js` — sticky nav active state, mobile menu, scroll reveal animations.
- `assets/` — profile placeholder and project visuals.
