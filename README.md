- Contact form removed.
- Contact section now keeps only Email, GitHub, and LinkedIn Connect.
- Resume download button is not included.

## GitHub Pages deployment

1. Unzip this folder.
2. Upload the files inside this folder to the root of the `hemanthkumarkonda.github.io` repository.
3. Make sure `index.html`, `styles.css`, and `script.js` are directly in the repo root.
4. Go to GitHub repo → Settings → Pages.
5. Set branch to `main` and folder to `/root`.
6. Open `https://hemanthkumarkonda.github.io/` after deployment completes.

## Files

- `index.html` — portfolio page content and SEO metadata.
- `styles.css` — layout, cards, timeline, spacing, and responsive design.
- `script.js` — active navigation, mobile menu, and scroll reveal animation.
