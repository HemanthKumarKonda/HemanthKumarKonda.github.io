# Hemanth Kumar Konda Portfolio

Static portfolio for GitHub Pages.

## Upload to GitHub Pages

1. Unzip this file.
2. Open the `hemanthkumarkonda.github.io` repository.
3. Upload these files directly into the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
4. Commit changes.
5. Open `https://hemanthkumarkonda.github.io/` after GitHub Pages finishes deployment.
